# Virtual Instrumentation Lab

A simulation-based virtual laboratory for studying the working principles of
classical flow and temperature measurement instruments.

## Instruments Covered
- Venturimeter
- Orifice Meter
- Rotameter
- Bimetallic Thermometer
- Liquid-in-Glass Thermometer

## Tech Stack
- Python
- NumPy
- Matplotlib
